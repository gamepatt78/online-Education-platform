Attendance Management

How to Manage Attendance

  • Go to the Attendance section from the sidebar menu.
  • Select the class and date for which you want to record attendance.
  • Mark each student as Present, Absent, or Late using the provided options.
  • Add remarks for individual students if needed (e.g., reason for absence).
  • Save the attendance record to update the student database.
  • Generate attendance reports for any class or date range as needed.
  • Export attendance data to PDF or Excel for record-keeping.

Tip: Regular attendance tracking helps monitor student participation and identify issues early.
